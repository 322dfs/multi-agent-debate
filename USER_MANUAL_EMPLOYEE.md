# 员工使用操作手册（上手版）

适用对象：公司员工（非开发/非运维）  
用途：把本系统当成“决策讨论 + 简历评审”工具来使用。

---

## 1. 打开系统

在公司内网浏览器输入：

- 系统地址：`http://192.168.108.150:43000`

打开后你会看到两个主要功能入口：

1. 智能辩论
2. 简历多 Agent 评审

---

## 2. 智能辩论怎么用

### 2.1 发起一场辩论

1. 点击“开始新辩论”
2. 输入辩题（例如：`AI 会取代程序员吗？`）
3. 选择场景（IT 决策 / 商业经营 / 人才成长等）
4. 选择辩手方式：
   - **手动选择**：自己勾选辩手
   - **智能推荐**：点击“智能推荐”按钮，让系统推荐
5. 点击“开始辩论”

### 2.2 看懂页面里的信息

- **辩论过程**：各辩手逐条发言
- **主持总结**：每轮观点冲突和重点问题
- **裁判结论**：阶段性建议与置信度
- **历史记录**：可回看之前的辩论

### 2.3 继续互动

- 可在输入框输入你的观点或质疑
- 可以选择“下一轮”继续辩论

---

## 3. 简历评审怎么用

### 3.1 基本流程

1. 点击“简历多 Agent 评审”
2. 选择岗位模板（或自定义岗位要求）
3. 上传简历（支持 PDF / DOCX / TXT / MD）
4. 可先点“仅解析简历（先看内容）”
5. 再点“开始评审”

### 3.2 评审结果包含什么

- 总结结论（推荐/有条件推荐/不推荐）
- 平均分
- 多个评审 Agent 的观点：
  - 亮点
  - 风险
  - 缺失项
  - 面试追问建议

### 3.3 历史评审

- 左侧“历史评审”可回看往期结果
- 可查看当次岗位、总结和评审意见

---

## 4. 推荐使用建议（给员工）

- 辩题越具体，结果越可执行  
  例如不要写“系统怎么优化”，建议写“我们是否要在 Q3 上线统一监控平台”
- 智能推荐不是强制结果  
  你可以改选辩手，保持人工判断
- 简历评审建议先解析  
  先确认文本提取正确，再正式评审

---

## 5. 隐私与数据说明（员工关心）

- 简历与评审数据保存在公司内部服务器
- 不会自动共享到外部互联网
- 请勿上传与工作无关的敏感个人文件

---

## 6. 常见问题（员工版）

### Q1：点了没反应

先刷新页面（`Ctrl + F5`），再重试。

### Q2：智能推荐为空

先确认已选择场景，再点击“智能推荐”。

### Q3：简历上传失败

先检查格式是否为 `pdf/docx/txt/md`。

### Q4：页面打不开

确认你在公司内网，并检查地址是否是：

`http://192.168.108.150:43000`

---

## 7. 反馈与支持

如果你遇到异常（页面报错、结果明显不合理、上传失败），请反馈给 IT 团队并附上：

1. 操作步骤
2. 报错截图
3. 大概时间点

这样能更快定位问题。
